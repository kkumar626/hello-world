# Claude Code Skills

| Skill | What it does | Scope |
|-------|-------------|-------|
| `/pr-description` | Writes a pull request description based on your branch changes | Global |
| `/commit-message` | Generates a conventional commit message and outputs the ready-to-run `git commit` command | Project-local |
| `/update-config` | Modify Claude Code settings — hooks, permissions, env vars, etc. | Built-in |
| `/review` | Reviews a pull request | Built-in |
| `/security-review` | Security review of pending changes on the current branch | Built-in |
| `/verify` | Runs the app and verifies a change actually works | Built-in |
| `/simplify` | Reviews changed code for quality and fixes issues found | Built-in |
| `/run` | Launches and drives the project app | Built-in |
| `/init` | Initializes a new CLAUDE.md file with codebase documentation | Built-in |
| `/fewer-permission-prompts` | Scans transcripts and adds allowlist rules to reduce permission prompts | Built-in |
| `/loop` | Runs a prompt or command on a recurring interval | Built-in |
| `/schedule` | Creates scheduled agents that run on a cron schedule | Built-in |
| `/claude-api` | Helps build, debug, and optimize Claude API / Anthropic SDK apps | Built-in |
| `/keybindings-help` | Customize keyboard shortcuts | Built-in |
