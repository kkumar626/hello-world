# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

A private sandbox repository used to test scripts and explore Claude Code features and courses.

## Commands

### Generate country data
```bash
python3 generate_countries_csv.py                   # outputs countries.csv (default)
python3 generate_countries_csv.py --format json     # outputs countries.json
python3 generate_countries_csv.py --format markdown # outputs countries.md
```

### Filter countries by continent
```bash
python3 filter_countries.py --continent Asia
python3 filter_countries.py --continent "North America" --input countries.json
python3 filter_countries.py --list-continents
```

## Architecture

**`generate_countries_csv.py`** — Single source of truth for all country data. The `COUNTRIES` list and `FIELDS` list are the only things to edit when adding data. Adding a new output format means adding a `write_<format>` function and registering it in the `FORMATS` dict.

**`filter_countries.py`** — Reads output files generated by `generate_countries_csv.py` (CSV or JSON). Auto-detects `countries.csv` if no `--input` is specified. Requires a `Continent` column to be present — always regenerate data files after modifying `generate_countries_csv.py`.

**`countries.csv` / `countries.json` / `countries.md`** — Generated files, not source of truth. Do not edit manually; regenerate via `generate_countries_csv.py`.
